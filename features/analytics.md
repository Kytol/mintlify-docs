# Analytics Dashboard

The Analytics Dashboard provides data-driven insights into platform usage, patient outcomes, and operational metrics through interactive visualizations.

## Overview

Analytics aggregates data across patients, carepaths, alerts, and activities to help care teams understand trends and optimize workflows.

## Time Period Selection

Filter all metrics by:
- Last 7 days
- Last 30 days
- Last 90 days
- This year

## KPI Cards

Four primary metrics with trend indicators:

### Metrics Displayed
- Total patients
- Active carepaths
- Completion rate
- Response time

### Card Features
- Large value display
- Trend arrow (up/down/stable)
- Percentage change vs. previous period
- Clickable for drill-down

## Activity Trends Chart

Bar chart showing patient interactions over time:
- Current period bars (cyan)
- Previous period comparison (gray)
- Hover tooltips with exact values
- Percentage change calculation

### Chart Summary
- Total activity count
- Average per period
- Peak day identification
- Overall trend percentage

## Patient Status Distribution

Horizontal bar chart showing patients by status:
- In Progress
- Awaiting Approval
- Completed
- On Hold

Each bar shows:
- Status label with color dot
- Patient count
- Percentage of total
- Clickable to filter patients

## Alert Severity Breakdown

Donut chart displaying alerts by severity:
- Critical (red)
- High (orange)
- Medium (yellow)
- Low (blue)

Center shows total alert count.

## Quick Stats Panel

Compact metrics grid:
- Icon and label
- Current value
- Color-coded by status

## Top Carepaths

Ranked list of most-used carepaths:
- Rank badge (gold/silver/bronze for top 3)
- Carepath name
- Patient count
- Completion rate with progress bar

"View All" opens full carepath modal.

## Response Time Metrics

Track care team responsiveness:
- Metric label and icon
- Current value
- Target comparison
- Progress bar
- Status indicator (good/warning/critical)

## Recent Activity Feed

Chronological list of platform events:
- Activity icon and type
- Description message
- Timestamp

## All Carepaths Modal

Expanded view of all carepaths:
- Full list with rankings
- Patient counts (total, completed, in progress)
- Completion rate bars
- Click to navigate to carepath

## Data Refresh

- Manual refresh button
- Loading indicator during refresh
- Auto-updates on period change

## Export

Export Report button generates downloadable analytics summary.

## Best Practices

1. Review analytics weekly for trends
2. Compare periods to identify changes
3. Monitor completion rates by carepath
4. Track response times for SLA compliance
5. Use filters to focus on specific timeframes
