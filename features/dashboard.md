# Dashboard

The Dashboard provides a comprehensive overview of the healthcare platform, displaying key metrics, upcoming tasks, and quick access to common actions.

## Overview

The dashboard serves as the home screen for healthcare providers, offering at-a-glance insights into patient care activities and easy navigation to frequently used features.

## Components

### Welcome Banner
Personalized greeting with contextual information about the current session.

### Quick Stats Widget
Four primary metrics displayed prominently:
- Total patients in the system
- Pending alerts requiring attention
- Completion rate percentage
- Average response time

### Metric Widgets
Six clickable cards showing real-time counts:
- Pending Approvals
- Active Carepaths
- Unread Messages
- Templates
- Questionnaires
- Alerts

Each widget navigates to its respective page when clicked.

## Quick Actions

Eight action buttons for common tasks:

| Action | Description |
|--------|-------------|
| Add Patient | Register a new patient |
| New Carepath | Create a carepath template |
| Questionnaire | Create a new form |
| Appointment | Schedule an appointment |
| Review Pending | Go to approval queue |
| Messages | View inbox |
| Analytics | View insights |
| Report Bug | Submit feedback |

## Upcoming Tasks

Scrollable list of scheduled tasks with:
- Task title and patient name
- Due time (relative or absolute)
- Priority indicator (high/medium/low)
- Task type badge (appointment, followup, medication, assessment)

### Task Actions
Click any task to open action modal:
- View Patient profile
- Mark Complete
- Reschedule
- Cancel Task

## Activity Feed

Real-time feed showing recent platform activity:
- Patient updates
- Form submissions
- Alert triggers
- System events

## Modal Forms

Quick actions open modal forms for:
- Creating carepath templates
- Building questionnaires
- Adding new patients
- Scheduling appointments
- Reporting bugs

## Navigation

Dashboard widgets provide quick navigation:
- Click any metric card to go to that section
- Quick action buttons open relevant pages or modals
- Task items link to patient profiles

## Best Practices

1. Check dashboard at shift start for overview
2. Address high-priority tasks first
3. Monitor pending approvals count
4. Review activity feed for recent changes
