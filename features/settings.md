# Settings

The Settings page allows users to customize their profile, adjust application preferences, and manage organization-level configurations.

## Profile Management

### User Profile Fields
- Avatar (URL-based image)
- Full Name (required)
- Email Address (required, validated)
- Role (read-only, admin-managed)

### Avatar Options
- Enter image URL
- Preview displayed in circle
- Remove button on hover
- Initials fallback when no image

### Validation
- Email format validation
- Required field indicators
- Error messages with icons
- Save confirmation feedback

## Website Settings

### Theme Options
- Dark (default)
- Light
- Blue

Theme changes apply immediately across the application.

### Font Size
- Small
- Medium (default)
- Large

Adjusts text size throughout the interface.

### Notifications
- Enable/disable in-app notifications
- Enable/disable email notifications

### Language
Supported languages:
- English
- Finnish
- French
- Spanish
- Swedish
- German

## Organization Settings

### Organization Selection
Switch between available organizations:
- Hospitals
- Departments
- Care Units

### Organization Details
View selected organization info:
- Organization type
- Address
- Phone number

### Switching Organizations
1. Select new organization from dropdown
2. Review organization details
3. Click "Switch Organization"
4. Settings persist across sessions

## Security Configuration

Admin-level settings (when available):
- Two-factor authentication requirements
- Password policy (length, complexity)
- Password expiry
- Login attempt limits
- Lockout duration
- IP whitelist
- Feature toggles (risk scoring, care team, A/B testing)

## Admin Configuration

Hospital-level settings:
- Hospital logo and name
- Primary color theme
- Support contact info
- Default timezone
- Session timeout
- Feature flags (messaging, AI features)
- Maintenance mode

## Data Persistence

Settings are saved to:
- Local storage for preferences
- Server for profile data
- Session for temporary state

## Best Practices

1. Keep profile information current
2. Choose appropriate theme for environment
3. Enable notifications for alerts
4. Review organization selection regularly
5. Use strong passwords and enable 2FA
