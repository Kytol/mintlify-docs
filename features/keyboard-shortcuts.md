# Keyboard Shortcuts

The Healthcare Platform supports keyboard shortcuts for efficient navigation and common actions, enabling power users to work faster without reaching for the mouse.

## Accessing Help

Press `?` at any time to open the keyboard shortcuts help modal.

## Navigation Shortcuts

Use `G` followed by a letter to navigate to different pages:

| Shortcut | Destination |
|----------|-------------|
| `G` then `D` | Dashboard |
| `G` then `P` | Patients |
| `G` then `A` | Analytics |
| `G` then `C` | Calendar |
| `G` then `M` | Messages |
| `G` then `S` | Settings |

Press `G`, then within 500ms press the second key.

## Action Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Open global search |
| `?` | Show keyboard shortcuts help |
| `Esc` | Close modal / Cancel action |
| `N` | Create new (context-aware) |

## Search Navigation

When global search is open:

| Key | Action |
|-----|--------|
| `↑` | Move selection up |
| `↓` | Move selection down |
| `Enter` | Select highlighted item |
| `Esc` | Close search |

## Modal Behavior

- `Esc` closes any open modal
- Click outside modal to close
- Tab navigation within modals

## Input Field Behavior

Keyboard shortcuts are disabled when:
- Typing in text inputs
- Typing in textareas
- Using select dropdowns

This prevents accidental navigation while entering data.

## Shortcut Help Modal

The help modal displays:
- Navigation shortcuts section
- Action shortcuts section
- Key combination display with `+` separator
- Footer reminder about `?` key

## Best Practices

1. Learn `Ctrl+K` for quick search access
2. Use `G` shortcuts for fast navigation
3. Press `?` when unsure of shortcuts
4. Use `Esc` to quickly close modals
5. Combine with mouse for hybrid workflow

## Customization

Shortcuts are currently fixed but designed around common patterns:
- `G` for "Go to" navigation
- `Ctrl+K` matches common search patterns
- `?` is intuitive for help
