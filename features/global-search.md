# Global Search

The Global Search feature provides instant access to patients, pages, and resources from anywhere in the application using a keyboard-driven interface.

## Activation

Open search with:
- Keyboard shortcut: `Ctrl+K` (or `Cmd+K` on Mac)
- Click search icon in navigation

## Search Interface

### Search Input
- Large, focused input field
- Placeholder text with search hints
- ESC key indicator for closing

### Results Display
- Grouped by type (patients, pages)
- Icon and title for each result
- Subtitle with additional context
- Type badge (patient, page, carepath, questionnaire)

## Search Capabilities

### Patient Search
Find patients by:
- Name (partial match)
- Patient ID

Results show:
- Patient name
- ID and current status
- Direct link to patient profile

### Page Navigation
Quick access to all main pages:
- Dashboard
- Patients
- Carepaths
- Questionnaires
- Alerts
- Calendar
- Messages
- Settings

## Recent Searches

When search is empty, displays:
- Recent search queries
- Associated results (if any)
- Clear all button
- Individual remove buttons

Recent searches are stored in session storage (cleared on browser close).

## Quick Links

Grid of common destinations when no search query:
- Dashboard
- Patients
- Carepaths
- Questionnaires
- Alerts
- Calendar
- Messages
- Settings

## Keyboard Navigation

| Key | Action |
|-----|--------|
| `↑` / `↓` | Navigate results |
| `Enter` | Select highlighted result |
| `Escape` | Close search |
| `Ctrl+K` | Toggle search |

## Visual Feedback

- Loading spinner during search
- "No results found" message
- Highlighted selected result
- Smooth animations

## Footer Help

Bottom bar shows:
- Navigation key hints
- Selection key hint
- Shortcut reminder

## Best Practices

1. Use `Ctrl+K` for fastest access
2. Type patient name or ID for quick lookup
3. Use recent searches for repeated queries
4. Navigate with keyboard for efficiency
